Introducing ShootDrop - a tool for creating Gear Lists for film shoots with a user-driven database of all the latest camera, lighting, grips and sound gear.

![My sick-ass landing page](./landing.png)

When I used to work on film shoots, I always struggled collaborating on creating gear lists. It freaking sucked.
During prep, I always sucked at making gear lists - they were time-consuming and hard to collaborate on.

![Speadsheets suck!](./spreadsheet.png)

With the affirmation of this issue and input from friends in the industry, I drafted up a list of the core issues that needed solving - building the blueprint for a lightweight, dead simple problem-centric solution.

![Excalidraw for the win](./)

I set out to implement this solution using NodeJS GraphQL API with a MongoDB database/AWS S3 and hooking into it with a NextJS web app complemented by TailwindCSS for rapid component styling along with Apollo Client for data fetching and state management.

Stoked to share that I'm ending the year with the MVP up and running! So far, we've had a number of filmmakers sign up and try it out since the launch over Instagram last week and I've been getting some awesome feedback on where improvements can be made... as well as some invaluable bug-crunching experience!
